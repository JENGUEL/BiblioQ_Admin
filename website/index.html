Enterprise IT Friendly

Built for School IT

BiblioQ is designed for straightforward deployment in school environments. Student data stays secure in a local SQLite database on each library PC — nothing is uploaded to our servers by default.

Install on Windows workstations with a standard installer, activate with a per-school license key, and keep every PC on the latest version through in-app auto-updates. IT teams can monitor installation health and manage licenses without disrupting school hours.

Security & Trust

Windows may warn you. Here's why BiblioQ is safe.

Defender and SmartScreen sometimes flag new software — especially tools used for remote IT support. BiblioQ is not malware; the warning is about reputation, not a virus scan failure.

BiblioQ uses Tailscale and OpenSSH so we can help your school remotely — fix issues without an on-site visit. Antivirus products treat those capabilities cautiously because bad actors abuse them too.

We have not purchased a commercial code-signing certificate yet ($200–400/year). Until download volume grows, Windows may show “Windows protected your PC.” Click More info, then Run anyway — the installer is safe.

  • Local SQLite database Student records stay on the library PC — not uploaded to our servers by default.
  • Remote tools are opt-in Tailscale and SSH are for IT remote support — configured only when your school enables the agent.
  • Code signing planned We will purchase an Authenticode certificate as soon as customer volume covers the annual cost.
