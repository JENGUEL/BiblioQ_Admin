Workflow 1

QR Attendance

  1. Librarian scans student QR code (printed on ID or card).
  2. BiblioQ logs the visit with timestamp.
  3. Dashboard updates "Today's Visitors" instantly.

Book Borrowing & Returns

  2. System checks borrowing limit and overdue status.
